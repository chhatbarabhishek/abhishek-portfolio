# Abhishek Chhatbar — Portfolio

Personal portfolio website for Abhishek Chhatbar. This is a responsive single-page portfolio built with HTML, CSS and JavaScript.

Live demo: https://abhishekchhatbar.netlify.app/

Features
- Responsive design
- Projects and contact information
- Simple single-page layout

Run locally
1. Clone the repo:

   git clone https://github.com/chhatbarabhishek/abhishek-portfolio.git

2. Open `index.html` in your browser or serve with a local dev server.

Set repository description & website (optional)

Use the GitHub web UI: go to your repository Settings → Repository name & description, and paste the live site URL into the "Website" field.

Or use the GitHub CLI (if installed and authenticated):

   gh repo edit chhatbarabhishek/abhishek-portfolio --description "Personal portfolio of Abhishek Chhatbar" --homepage "https://abhishekchhatbar.netlify.app/"
